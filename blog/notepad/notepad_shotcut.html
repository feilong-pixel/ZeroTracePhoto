SCI_UNDO（撤销上一次操作） Ctrl + Z or Alt + Backspace 编辑
SCI_REDO（恢复上一次操作） Ctrl + Y or Ctrl + Shift + Z 编辑
SCI_NEWLINE（新行） Enter or Shift + Enter 编辑
SCI_TAB（制表键） Tab 编辑
SCI_BACKTAB（回退制表键） Shift + Tab 编辑
SCI_ZOOMIN（放大） Ctrl + Num + 编辑
SCI_ZOOMOUT（缩小） Ctrl + Num - 编辑
SCI_SETZOOM（缩放重置） Ctrl + Num / 编辑
SCI_SELECTIONDUPLICATE（行拷贝） Ctrl + D 编辑
SCI_EDITTOGGLEOVERTYPE（插入覆盖变换） INS 编辑
SCI_LINEDOWN（下一行） Down 编辑
SCI_LINEDOWNEXTEND（向下选择一行） Shift + Down 编辑
SCI_LINEDOWNRECTEXTEND（块选择向下移动一行） Alt + Shift + Down 编辑
SCI_LINESCROLLDOWN（向下滚动一行） Ctrl + Down 编辑
SCI_LINEUP（向上移动） Up 编辑
SCI_LINEUPEXTEND（向上选择一行） Shift + Up 编辑
SCI_LINEUPRECTEXTEND（块选择向上移动一行） Alt + Shift + Up 编辑
SCI_LINESCROLLUP（向上滚动一行） Ctrl + Up 编辑
SCI_PARADOWN（移动到段落末） Ctrl + ] 编辑
SCI_PARADOWNEXTEND（选择到段落末） Ctrl + Shift + ] 编辑
SCI_PARAUP（移动到段落首） Ctrl + [ 编辑
SCI_PARAUPEXTEND（选择到段落首） Ctrl + Shift + [ 编辑

※备注：

  1. 在使用快捷键时要注意，对于单词的选择移动，是不支持中文的。

   

※提示：

  1. 《主菜单》快捷键，可以在左面的菜单里面变更：（菜单）设置 → 管理快捷键 → 主菜单（页面）
  2. 《Scintilla命令》快捷键，可以在左面的菜单里面变更：（菜单）设置 → 管理快捷键 → Scintilla命令（页面）
