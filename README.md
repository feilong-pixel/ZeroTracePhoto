# ZeroTracePhoto

Local-first photo organizer. No upload. No cloud.

local-first photo browser and organizer designed for safely managing large image collections on your own computer.

ZeroTraceBrowser 是一个本地优先的轻量图片浏览与整理工具，专为需要安全管理大量本地照片的用户设计。照片不上传云端，所有整理操作都在你的电脑上完成。

ZeroTraceBrowser は、ローカル環境で写真を安全に閲覧・整理するための軽量な画像管理ツールです。写真をクラウドへアップロードせず、すべての操作を自分のPC内で完結できます。
