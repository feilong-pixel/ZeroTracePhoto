Built for controlled local browsing, duplicate detection, safe delete, and batch organization.

A simple tool for serious photo organization.

ZeroTraceBrowser focuses on the essential workflow: browse, inspect, copy, detect duplicates, and safely remove files.

💻

Local-first

Photos stay on your computer. No cloud upload is required.

⚡

Lightweight

Designed as a focused browser and organizer, not a heavy media suite.

🔍

Duplicate detection

Find visually similar images with perceptual hash based comparison.

♻️

Safe delete

Move files to a controlled recycle area instead of deleting them directly.

🖼️

Image viewer

Preview images and inspect useful metadata such as EXIF information.

🌐

Multi-language UI

Who is it for?

For users who want practical local photo management without giving up control of their files.

Photographers and hobbyists Quickly browse and clean up large local photo folders.
Engineers and power users Use a transparent local tool with controlled file operations.
Privacy-conscious users Organize personal images without uploading them to a third-party cloud.

Get started locally.

Clone the repository, install dependencies, start the server, and open it in your browser. 

git clone https://github.com/feilong-pixel/ZeroTraceBrowser.git
cd ZeroTraceBrowser

pip install -r requirements.txt

python app.py

# Open:
# http://127.0.0.1:8000

Your photos stay with you.

ZeroTraceBrowser is designed around local-first usage. It helps you browse and organize files on your own machine with explicit, controlled operations.

No cloud upload

The tool is intended to run locally on your computer.

Controlled actions

Copy, delete, and batch operations are designed to remain visible and intentional.
